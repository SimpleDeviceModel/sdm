This directory contains example SDM plugins.

"simpleplugin" is written in C and demonstrates only basic SDM features.
Register and data streams are simulated in software.

"uartdemo" communicates with an Arduino Uno board over the serial port and
acts like a virtual oscilloscope. It is written in C++. See the corresponding
readme for details.

"testplugin" is a software simulated test plugin used by the SDM test suite.
It is also written in C++.
